# Football Market Value vs Age (Python)

## Objective
Analyze how football players' market value evolves with age using a public dataset.

## Key question
How does the average market value change across ages?

## Tools
Python, pandas, matplotlib (and/or seaborn)

## Project structure
- `notebooks/`: analysis notebook
- `src/`: helper functions (cleaning, plotting)
- `data/`: dataset (or instructions to download it)
